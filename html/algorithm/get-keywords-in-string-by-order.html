将关键字(多个)从一个字符串中按顺序提取出来

2017-05-22

(#a)-?-(#b);(#b)-适用->(#a)<-适用?-(#x1)<-包含?-(#b)UNION(#b)-禁忌|慎用->(#a)<-禁忌|慎用?-(#x1)<-包含?-(#b)UNION(#a)<-适用-(#功效:x1)<-具有?-(#b);(#b)-导致?->(#x1)-属于|从属->(妇科病),搜索";","UNION",得到列表结果为：(;,UNION,UNION,;)

简单算法:

							String expression = "(#a)-?-(#b);(#b)-适用->(#a)<-适用?-(#x1)<-包含?-(#b)UNION(#b)-禁忌|慎用->(#a)<-禁忌|慎用?-(#x1)<-包含?-(#b)UNION(#a)<-适用-(#功效:x1)<-具有?-(#b);(#b)-导致?->(#x1)-属于|从属->(妇科病)";
							List keywords = new ArrayList<>();
							int location1 = expression.indexOf(";", 0);
							int location2 = expression.indexOf("UNION", 0);
							int fromIndex =  0;
							while(location1 > 0 || location2 > 0){
								if(location1 < 0 && location2 > 0){
									//没有;，有UNION；
									keywords.add("UNION");
									fromIndex = location2 + 5;
								}else if(location1 > 0 && location2 < 0){
									//有;，没有UNION
									keywords.add(";");
									fromIndex = location1 + 1;
								}else{
									//都有，找小的下标，将对应的字符串加入到列表，将开始查找位置设为小的那个
									if(location1 < location2){
										keywords.add(";");
										fromIndex = location1 + 1;
									}else{
										keywords.add("UNION");
										fromIndex = location2 + 5;
									}
								}
								location1 = expression.indexOf(";", fromIndex);
								location2 = expression.indexOf("UNION", fromIndex);
							}

						String expression = "(#a)-?-(#b);(#b)-适用->(#a)<-适用?-(#x1)<-包含?-(#b)UNION(#b)-禁忌|慎用->(#a)<-禁忌|慎用?-(#x1)<-包含?-(#b)UNION(#a)<-适用-(#功效:x1)<-具有?-(#b);(#b)-导致?->(#x1)-属于|从属->(妇科病)";
						List keywords = new ArrayList<>();
						Matcher matcher = Pattern.compile(";|UNION").matcher(expression);
						while (matcher.find()) {
							keywords.add(matcher.group());
